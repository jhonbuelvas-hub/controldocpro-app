# ControlDocPro

Software de gestión documental y contractual en la nube.

## Objetivo
Centralizar la gestión de contratos, documentos, correspondencia y controversias, permitiendo control, trazabilidad y cumplimiento.

## Módulos principales
- Usuarios y roles
- Terceros
- Contratos
- Documentos
- Correspondencia
- Obligaciones
- Controversias
- Alertas
- Reportes

## Estado del proyecto
En desarrollo (MVP inicial)

## Tecnologías
- Backend: Python
- Base de datos: PostgreSQL
- Hosting: Render
- Repositorio: GitHub
